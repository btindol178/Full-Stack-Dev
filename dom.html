You will use javascript file to change the size of the header, but first we need to learn how to use JavaScript to actually grab the HTML objects from the DOM.

Here are some important document attributes:

  • Document.URL -- This is actually url of website
  • Document.body -- This is actually everything inside body
  • Document.head -- Tis is everything in the head of page
  • Document.links -- There are all the links on the page

Then we also have methods we can use to grab HTML elements:

  • Document.getElementById() -- Returns the element with the id
  • Document.getElementByClassName() -- Returns list of all elements beloging to Document.getElementByClassName
  • Document.getElementsByTagName() -- Returns list of all elements with the tag
  • Document.querySelector() -- Returns the first object matching the CSS style Document.querySelector
  • Document.querySelectorAll() -- Returns all objects matchin the CSS style selector

The first three of these are pretty selfe explanatory. The querySelector() can accept CSS style selector, so you can use # or. if you want ids or classes. basically a more generalized method where you can type any CSS style.
